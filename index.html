Smart Flower Atlas

powered by AiR(AI+AR)

Advanced Image Recognition System

See the workflow for developer

A smart flower atlas for Daan Forest Park in Taipei, built on open source image recognition systems, powered by advanced artificial intelligence (AI) and Augmented Reality (AR) technologies.

Initial Proposal Written in Chinese

Features

  • Data collection with viewing angles and AR scale
  • On-device realtime flower identification (it's extremely fast)
  • Open source and open data
